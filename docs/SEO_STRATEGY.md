# SEO Strategy

## Primary Objective
Build a strong Persian-first foundation for long-tail utility search while also indexing full English equivalents from day 1.

## Core SEO Model
- language-first routing
- category pages
- tool detail pages
- built-in tool pages
- redirect landing pages
- coming-soon pages for future demand capture

## Route Strategy
- `/fa/`
- `/en/`
- `/fa/categories/[slug]`
- `/en/categories/[slug]`
- `/fa/tools/[slug]`
- `/en/tools/[slug]`

## Canonical Strategy
- each language page has self-canonical
- Persian pages treated as primary market, not as canonical replacement of English
- use hreflang cross-linking between equivalents

## Indexable Page Types
1. Homepage per language
2. Category pages
3. Tool pages
4. Redirect tool pages
5. About and trust pages
6. Select coming-soon pages with useful copy

## Tool Page Content Blocks
Each tool page should include:
- clear H1
- concise utility intro
- how it works
- use cases
- FAQ
- related tools
- alternatives
- privacy or workflow notes if relevant

## Coming Soon Page Rules
A coming-soon page must not be thin.
It should include:
- what the tool/category will do
- who it is for
- example use cases
- related available tools
- alternatives or adjacent tools
- internal links

## Metadata Requirements
- unique title per page
- unique meta description per page
- OG title/description/image
- hreflang alternates
- structured breadcrumb data
- software/application schema where relevant
- website schema on homepage
- collection/list schema on category pages

## Internal Linking Strategy
- homepage -> top categories
- homepage -> featured tools
- category -> tools
- tool -> related tools
- built-in -> alternatives -> category
- coming-soon -> related live tools

## Programmatic SEO Direction
Post-MVP expansion can generate intent pages around:
- task + file format
- tool + audience
- tool + language
- best tool for use case
- alternatives

## Content Quality Rule
Avoid empty SEO pages.
Every indexable page must provide utility, explanation, trust, and navigation value.
