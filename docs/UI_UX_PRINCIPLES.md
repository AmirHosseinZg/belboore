# UI/UX Principles

## Experience Goal
Belboore should feel like:
- a calm work surface
- a useful digital desk
- a dependable utility hub
- familiar, efficient, and welcoming

## Core Principles
1. Utility first
2. Search first
3. Friendly clarity
4. Fast interaction
5. Low cognitive load
6. Strong visual hierarchy
7. Persian-native comfort
8. Desktop excellence with solid mobile usability

## Homepage Experience
The homepage must:
- feel productive and alive
- help users resume work quickly
- support browsing and direct search
- encourage repeat usage

## Interaction Rules
- primary actions obvious
- tool cards clean and informative
- compact but readable metadata
- recent tools visible and easy to reuse
- avoid clutter and dashboard overload

## Visual Feel
- minimal
- slightly premium
- clean surfaces
- controlled accent use
- soft shadows
- rounded but not childish
- subtle motion only where helpful

## Accessibility
- keyboard usable
- strong focus states
- contrast-safe in both themes
- clear labels and helper text
- responsive layout without hidden utility loss

## Empty and Coming Soon States
These should feel intentional, not broken.
They must include:
- explanation
- adjacent alternatives
- invitation to explore related tools
