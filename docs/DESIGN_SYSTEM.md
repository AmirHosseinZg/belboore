# Design System

## Brand Personality
- capable
- friendly
- modern
- trustworthy
- lightweight

## Base Palette
- Primary / Tech Blue: `#0052FF`
- Accent / Electric Cyan: `#00D4FF`
- Light Canvas / Ice-Slate: `#FAF9FC`
- Dark Canvas / Deep Navy: `#0B132B`
- Dark Card / Slate Card Base: `#1C2541`
- Deep Base / Obsidian Base: `#02040A`

## Suggested Extended Palette
### Light theme
- Background: `#F7FAFF`
- Surface: `#FFFFFF`
- Surface Alt: `#F1F5FB`
- Border: `#D8E2F0`
- Text Primary: `#10203A`
- Text Secondary: `#52627A`

### Dark theme
- Background: `#0B132B`
- Surface: `#111A36`
- Surface Alt: `#1C2541`
- Border: `#2A3A63`
- Text Primary: `#F4F7FC`
- Text Secondary: `#A9B7D0`

## Typography
Use highly readable web-safe or modern system-forward stack with Persian support.

Recommended:
- Persian: Vazirmatn
- English/UI: Inter
- Fallbacks: system-ui, sans-serif

## Radii
- small: 10px
- medium: 16px
- large: 24px

## Shadows
Use light, soft shadows in light theme and subtle depth borders in dark theme.

## Component Inventory
- App shell
- Header
- Search bar
- Tool card
- Category card
- Featured panel
- Recent tools panel
- CTA button
- Tag chips
- FAQ accordion
- Tool workspace panel
- Empty state
- External redirect panel
- Footer

## Motion
- fast hover feedback
- subtle elevation
- theme transition optional but light
- no heavy motion blocking usability
