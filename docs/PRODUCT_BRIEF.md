# Product Brief

## Product Name
Belboore

## Persian Name
بل بوره

## Positioning
Persian: جعبه ابزار دیجیتال شما در یکجا، رایگان و متن‌باز

English: Your digital toolbox in one place — free, open-source, and built for everyday work.

## Product Type
Hybrid platform:
- built-in tools
- curated outbound tools

## Primary Audience
General users.

## Core Promise
- all-in-one convenience
- free
- open source
- easy to use
- bilingual
- privacy-conscious where possible

## MVP Goal
Launch a polished, SEO-strong frontend MVP that:
- introduces the Belboore brand
- establishes category and tool page architecture
- delivers 5 useful built-in client-side tools
- supports curated outbound tool discovery
- is deployable today

## Non-Goals for MVP
- backend-dependent tools
- user accounts
- payments
- user submissions
- full admin panel
- blog
- complex analytics backend
