# Information Architecture

## Main Navigation
- Home
- Categories
- Featured Tools
- About

Secondary:
- Language switch
- Theme switch
- Search

## Homepage Sections
1. Hero search and greeting
2. Quick categories
3. Recently used tools
4. Featured built-in tools
5. Featured external tools
6. Curated collections
7. Trust/open-source section
8. SEO utility content block

## Core Page Types
1. Homepage
2. Category page
3. Tool page
4. Redirect tool page
5. Legal page
6. About page

## Navigation Principles
- tools should be reachable in 2-3 interactions
- homepage must feel useful immediately
- search should be prominent
- categories should be visually scannable
- related tools should increase depth and retention

## Tool Page Structure
1. title and short description
2. actual tool interface
3. quick instructions
4. trust or privacy note if relevant
5. FAQ
6. alternatives
7. related tools

## Category Page Structure
1. category title and intro
2. featured tools
3. all tools grid
4. coming soon tools
5. FAQ or explanatory block
6. related categories

## Redirect Page Structure
1. title and description
2. what the tool is for
3. why it is useful
4. go-to-tool CTA
5. related alternatives
6. note if external
